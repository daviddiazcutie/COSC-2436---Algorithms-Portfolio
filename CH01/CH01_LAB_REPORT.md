## Algorithm Summary
- **How it works:** Binary search works by repeatedly halving the search space. Given a sorted array, it checks the middle element and determines whether the target is in the left or right half, discarding the other half and repeating until the element is found or the search space is empty.
- **Time complexity:** O(log n) for binary search vs. O(n) for linear search.
- **When to use it:** Binary search is used when searching a sorted collection and fast lookup is needed. Linear search is used when the collection is unsorted or too small to warrant sorting first.
